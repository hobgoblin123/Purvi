# Product Sync — Instructions

## What this folder is

`Product sync/` holds notes from product sync meetings. Each file is a dated record of what was discussed, decided, and planned.

## File naming

- Files are named by date: `DD-MM-YYYY.md`
- One file per day. If multiple syncs happen on the same day, append to the existing file — do not create a second file with the same date.
- When appending a second sync on the same day, add a horizontal rule (`---`) separator and note the time in `HH:MM` format as a subheading.

## File structure

Each sync note starts with the date as a heading, followed by the discussion points, decisions, and any action items that came out of the conversation.

## Example

```
# 08-06-2026

## Sync — 16:30

- Discussed X
- Decided Y
- Action: Z

---

## Sync — 21:00

- Follow-up on X
- New topic W
```

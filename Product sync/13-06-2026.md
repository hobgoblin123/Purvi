# 13-06-2026

## Sync

Scope expansion for finance_sheet — two new on-demand views, plus a new During-building gate.

### New features scoped

**1. Historic Net Position chart**
- Tap Net Position card → overlay with line chart
- One data point per salary cycle, full FY
- Simple view — just the line chart with a visible zero line, no cycle/custom toggle, no category breakdown
- Y-axis shows full range (how low/how high)
- Lazy-loaded, reuses recharts from spending history
- New endpoint: `GET /api/net-position-history`

**2. Credit Card spend drilldown**
- Tap a credit card mini-card → overlay with transaction list for that card's statement cycle
- Statement cycle uses the card's own statement day (e.g. 25th-to-25th for Amazon Pay)
- Prev/next navigation across statement cycles
- Each transaction tappable → EditSheet (same pattern as spending history)
- Lazy-loaded
- New endpoint: `GET /api/card-transactions?card=<name>&cycle=<offset>`

### Design decisions

- Separate endpoints for both features (dashboard payload already large, these are on-demand)
- Interaction model follows spending history's pattern (overlay, tap-to-edit) but Net Position keeps it simpler (chart only, no tabs/toggles)
- Both chunks lazy-loaded per the new "Defer what isn't visible" gate

### New gate

**During building → Defer what isn't visible** — lazy-load on-demand views; don't pay the bundle cost for features the user might not open.
